# Frontend - User Directory

This folder contains a React application that displays a user directory with search, sort, pagination, and filter functionalities.

## Setup and Running

1.  **Install Dependencies:**
    Open a terminal in the `/frontend` directory and run:
    ```bash
    npm install
    ```

2.  **Start the Application:**
    To start the application, run the following command:
    ```bash
    npm start
    ```
    The application will be running at `http://localhost:3000`.

## Features

-   **Search:** Search for users by their first name or email.
-   **Sort:** Sort the user list by first name or email.
-   **Pagination:** Navigate through the pages of user data.
-   **Filter:** Filter users by their email domain.
-   **Loading Spinner:** A loading spinner is displayed while the user data is being fetched.
-   **Mobile Responsive:** The application is designed to be responsive on mobile devices.
